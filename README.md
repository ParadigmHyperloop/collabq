# Forum App

### Getting Started

1. `bundle install`
2. `rake db:setup`
3. `rails s`

### Linters

Always use linters to keep the code clean.

* Ruby - Rubocop
* Haml - haml_lint
* JS - ESLint
* SASS - sass-lint
